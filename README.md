# Research Publish Shop 📚

A responsive book publishing and shop UI built with Next.js and Tailwind CSS.  
This project is a frontend clone based on a Figma design.

 🚀 Tech Stack
- Next.js
- React
- Tailwind CSS
- Swiper.js


 ✨ Features
- Responsive design
- Book slider using Swiper
- Modern UI based on Figma
- Reusable components


 ⚙️ Setup

npm install  
npm run dev  

Open: http://localhost:3000
